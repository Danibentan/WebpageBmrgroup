# Fotos de productos de tienda

Subí acá las imágenes reales de los productos de `/tienda` en formato `.webp`.

La convención configurada en `lib/products.ts` es:

```txt
<slug>-portada.webp
<slug>-1.webp
<slug>-2.webp
<slug>-3.webp
```

Ejemplo para Ventana Corrediza A40:

```txt
ventana-corrediza-a40-portada.webp
ventana-corrediza-a40-1.webp
ventana-corrediza-a40-2.webp
ventana-corrediza-a40-3.webp
```

En la app estas imágenes se sirven como `/productos/<archivo>.webp`.
